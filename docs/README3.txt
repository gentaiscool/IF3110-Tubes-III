How to use this Chintalian(Chinese-Italian)website :
1. Pastikan tomcat versi 7.0 sudah terinstall dengan baik.
2. Buat database pada localhost/phpmyadmin dengan nama 'chintalian'.
3. Import 'chintalian_db.sql' pada database anda.
4. Buka browser lalu ketik localhost:8080/Chintalian
5. Silakan register jika belum mendaftar, silakan login jika sudah mendaftar!
6. Enjoy IT!! :3
